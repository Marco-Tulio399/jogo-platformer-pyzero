#Platformer - pyzero
Este é um jogo platformer criado em python, o objetivo do jogo é pegar todas as bandeiras com sem ser pego pelos inimigos.
Para controlar o seu personagem você utiliza as setas direita e esquerda ou a tecla "a" e "d" e a tecla espaço para pular.
O jogo foi criado com auxílio da biblioteca pyzero do python.
